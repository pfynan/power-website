Power Analyzer

A better android power analysis app

About

Power Analyzer is an app that more accurately estimates the power being used by and Android phone. It improves upon existing methods by measuring current directly to give more accurate real-time estimates, using additional variables to model the power such as performance counters, and by reporting on combinations of apps that together take more power than each does alone.

How It Works

Power Analyzer continuously updates a model of the phone's power consumption using real-time information from the current sensor. The model takes information about which modules are being used on the phone and to what degree and uses it to predict the power being used. Using the model, Power Analyzer compares each app's contribution to the total power consumption of the phone to identify which apps cause the phone to consume much more power
